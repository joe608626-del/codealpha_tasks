Task 2 – High-Fidelity UI Design

Project Title

Farmer to Customer Direct Integration System

Overview

This project was created as part of the CodeAlpha UI/UX Design Internship. The objective of this task is to convert the low-fidelity wireframe into a high-fidelity UI design with proper colors, typography, icons, and visual elements to create a realistic and user-friendly mobile application interface for direct farmer-to-customer interaction.

Design Tool

- Figma

Prototype

Figma Prototype Link: https://www.figma.com/proto/EdEFY2YPnbja7H98WISq3m/Task_2_High_fidelity_UI?node-id=0-1&t=LE7Lz06OnAsOFKtE-1

UI Screens Included

- Splash Screen
- Welcome / Onboarding Screen
- Farmer Login & Registration
- Customer Login & Registration
- Home Dashboard
- Product Listing Screen
- Product Details Page
- Shopping Cart
- Checkout / Order Confirmation
- Order Tracking Screen
- User Profile Screen
- Settings Screen
- Logout Functionality Screen

Key Features

- Direct farmer-to-customer product browsing system
- Real-time order tracking for customers
- Secure login and registration system for both user types
- Easy cart and checkout flow
- User profile management
- Settings for app preferences and account control
- Simple logout functionality for session security

Design Highlights

- Clean and modern UI design
- Agriculture-inspired color palette (green, natural/earth tones)
- Consistent typography and spacing system
- Intuitive navigation and user flow
- Reusable UI components for scalability
- Mobile-first responsive design approach

User Flow

Welcome → Select User Type → Login/Register → Home Dashboard → Browse Products → Product Details → Add to Cart → Checkout → Order Tracking → Profile → Settings → Logout

Deliverables

- High-Fidelity UI Design PDF
- Interactive Figma Prototype

Objective

To design a complete, user-friendly mobile application interface that improves the experience of farmers and customers by enabling direct product selling, order tracking, and account management.
