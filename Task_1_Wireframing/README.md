Task 1 – Wireframing & Low-Fidelity Design

Project Title

Farmer to Customer Direct Integration System

Overview

This project was created as part of the CodeAlpha UI/UX Design Internship. The objective of this project is to design a low-fidelity wireframe for a mobile application that connects farmers directly with customers. The application helps farmers sell their products without intermediaries while allowing customers to purchase fresh produce at fair prices.

Design Tool

- Figma

Prototype

Figma Prototype Link: https://www.figma.com/proto/s4iNcuW5K1HcwB5oLQl2wN/Task_1_Wireframing?node-id=0-1&t=vKjfMvWxsZ7Kyhyz-1

Features Included

- Welcome Screen
- Farmer Registration & Login
- Customer Registration & Login
- Home Dashboard
- Product Listing
- Product Details
- Shopping Cart
- Order Placement
- User Profile

User Flow

Welcome Screen → Select User Type → Farmer Login / Customer Login → Home → Browse Products → Product Details → Add to Cart → Place Order → Order Confirmation

Deliverables

- Wireframe Design (PDF)
- Interactive Figma Prototype

Objective

To create a simple, user-friendly, and intuitive wireframe that enables direct interaction between farmers and customers, ensuring a smooth buying and selling experience.
