Task 3 – UX Case Study

Title

UX Case Study: Paytm – Digital Payments User Experience Evaluation

Submitted By

Jyoshna Goluguri
Department: Artificial Intelligence & Data Science

Overview

This UX case study evaluates the Paytm application to understand its usability, navigation structure, and user experience challenges. The goal is to identify key issues and suggest improvements for a more simple, accessible, and user-friendly design.


Objective

- Evaluate Paytm usability
- Identify UX issues
- Improve navigation and accessibility
- Enhance user satisfaction


Key Findings

- Difficulty finding frequently used features
- Homepage clutter due to promotions
- Limited personalization
- High cognitive load for new users


Recommendations

- Voice-assisted navigation
- Simplified quick access mode
- Personalized dashboard


Conclusion

Paytm is a powerful platform, but UX improvements like simplification, personalization, and accessibility enhancements can make it more user-friendly for all types of users.
